Get Up

Get Up is geared towards teenagers who need unique inspiration based on their different moods. After selecting how they feel, the users are presented with the option of receiving inspiration in the forms of songs, quotes and videos. Get Up was created during Qeyno Labs' My Brother’s Keeper Hackathon the weekend of February 26th, 2016. It received the 3rd place prize.

Technologies used: Javascript, jQuery, Bootstrap, CSS and HTML.




Vibart


Vibart allows for buyers to search local artists in major cities to find creators to either purchase a piece or order a custom work. A portfolio is generated by accessing each artist's instagram to give a quick insight into their style and vibe. Once clients work with artists, they have the opportunity to review their experience.

Ruby on Rails, PostgreSQL, bcrypt, bootstrap and Instagram’s API were used to create this project.



atCon


atCon is a web app designed to help convention and tradeshow attendees network through social events. Users will be able to create an account, log into their account, create events to host, and select events they want to go to during their conference.

Technologies used: Ruby on Rails, Bootstrap, jQuery, jQuery Validation, AJAX, Paperclip, AWS SDK, RSpec, LinkedIn API, Bcrypt, PostgreSQL, GitHub, and Heroku.
